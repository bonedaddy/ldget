# Argu Command Line Interface

A simple command line interface tool to get items from [argu.co](https://argu.co).

## Run locally
`git clone https://bitbucket.org/joepio/argu-cli`
`go run ./argu.go get`

Or compile and run:
`go build`
`./argu-cli get`

Or install to `bin` and run:
`go install`
`argu-cli get`
